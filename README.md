# ft_ls

This project consists in coding ls UNIX command.

## Compiling

Run with make.

## Usage

./ft_ls [-ARSTacdlprtu] [file ...]

## Overview

* Basic Flag:

-l, -R, -a, -r, -t.

* Bonus Flag:

-A, -S, -T, -c, -d, -p, -u.

## Objectives

* Tree of files and directories.

* Types and characteristics of files.
